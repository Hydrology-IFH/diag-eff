Diagnostic efficiency – specific evaluation of model performance
================================================================

`diag-eff` is a package for diagnostic evaluation of hydrologic simulations. It
includes one main module `de`. The module provides functions for calculating
model performance and for diagnostics.


How to cite
===========

In case you use de in other software or scientific publications,
please reference this package. It is published and has a DOI. It can be cited
as:

    Robin Schwemmle, Dominic Demand & Markus Weiler (2021). diag-eff 1.1:
    Diagnostic efficiency – specific evaluation of model performance.
    (Version v1.1). Zenodo. https://doi.org/10.5281/zenodo.4590174
